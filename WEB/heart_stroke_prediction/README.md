# Heart Stroke Prediction

This small Flask app predicts the likelihood of a stroke using a KNN model and a StandardScaler.

Files:
- `app.py` - Flask backend that serves `templates/index.html` and `/predict` endpoint.
- `templates/index.html` - Frontend form to enter features and show results.
- `train_model.py` - Simple script to train a KNN model and scaler from `stroke_prediction_custom_dataset.csv`.
- `stroke_prediction_custom_dataset.csv` - Dataset used to train the model.
- `model.pkl`, `scaler.pkl` - Model artifacts (generated by `train_model.py`).

Quick start (Windows cmd.exe):

1. Create a virtual environment (recommended):

```powershell
python -m venv .venv
.\.venv\Scripts\activate
```

2. Install requirements:

```powershell
pip install -r requirements.txt
```

3. Train the model (this will create `model.pkl` and `scaler.pkl`):

```powershell
python train_model.py
```

4. Run the app:

```powershell
python app.py
```

5. Open http://127.0.0.1:5000 in your browser and use the form.

Notes:
- `train_model.py` includes heuristics to map common CSV column names to the fields used in the frontend. If your dataset uses different names, edit the script accordingly.
- The app will return a JSON with `result` ("Stroke" / "No Stroke") and `probability` when available.
